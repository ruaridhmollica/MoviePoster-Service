# MoviePoster-Service
A web service that provides information about films using an API:
* [Movie Poster Service](http://www2.macs.hw.ac.uk/~rm141/LAB5/theMovieService.html)
#
# Features:
* Retrieves movie title
* Rating
* Director
* Summary
* Poster
